This was a project that I started in 2016, when I was taking calculus in high school. I figured that if I spent so long on calculus homework, I should create a program that would do it for me. Instead of finishing the project in time to use it, I managed to finish three semesters of calculus before I even managed to get the program to do simple derivations. But it worked! Eventually.

The way it worked was by parsing equations as trees and then substituting sections of the equation tree until it was sufficiently simplified. For example, imagine that you wanted to find the derivative of 3x^2 with respect to X. This would produce the equation tree:

I've marked all the X's as variables by appending "_VAR" to them. This is both how the parser flags X as a variable, and because we need to distinct between different types of variables, which you'll see why later. Now, for this example we want to simplify the equation in order to get rid of the derivation operator.

Fortunately, mathematicians have done a lot of work done for us by already figuring out that some mathematical equations are actually the same as each other. As a really simple example, we know that $$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$
