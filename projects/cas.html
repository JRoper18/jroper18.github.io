This was a project that I started in 2016, when I was taking calculus in high school. I figured that if I spent so long on calculus homework, I should create a program that would do it for me. Instead of finishing the project in time to use it, I managed to finish three semesters of calculus before I even managed to get the program to do simple derivations. But it worked! Eventually.

Being my largest project, I learned a lot about Test-Driven-Development and JUnit, which has been the single thing that has really made programming easier and more enjoyable. I also learned how to keep large codebases maintainable, so that changing one thing wouldn't result in the entire codebase crashing down.
